Smart Management System
Project Description
A simple Smart Management System login application developed using HTML, JavaScript, Selenium WebDriver, and Java. The project automates login validation, executes test cases, and verifies alert messages for successful and unsuccessful login attempts.
Technologies Used
HTML
JavaScript
Java
Selenium WebDriver
Features
Username and Password Validation
Login Success and Failure Alerts
Selenium Automation Testing
Alert Handling
